# Stager — инструкция для пользователя

Stager — это бот в Telegram, который помогает собирать траты по каждому проекту, чтобы в конце получить готовую таблицу.

## Как начать

1. Открой invite-ссылку: `t.me/<bot_username>?start=...`
2. Бот напишет «Добро пожаловать». Всё, ты зарегистрирован(а).

## Создать проект

Напиши команду:

```
/newproject
```

Бот спросит название и бюджет (бюджет можно пропустить). Новый проект автоматически становится активным.

## Записать трату

### Способ 1 — фото чека (самый быстрый)

Просто отправь боту фотку чека. Через несколько секунд он пришлёт карточку:

> 🧾 ИКЕА
> 💰 4 850 ₽
> 📅 23.05.2026
> 🏷 мебель
>
> [✅ Сохранить] [✏️ Сумма] [✏️ Категория] [✏️ Вендор] [❌ Отмена]

Если всё правильно — жми **Сохранить**. Если нужно поправить — жми соответствующую кнопку.

### Способ 2 — текстом

```
/add 4850 мебель диван из ИКЕА
```

Формат: `/add <сумма> <категория> <описание>`

Категории: мебель, декор, текстиль, доставка, бригада, расходники, фото, аренда, транспорт, прочее.

## Если у тебя несколько проектов одновременно

```
/list      — показать все мои проекты
/switch    — переключить активный
```

## Получить итоговую таблицу

В конце проекта:

```
/report
```

Бот пришлёт краткую сводку (сколько всего, по категориям) и кнопку «📥 Скачать таблицу». Жми — получишь файл Excel со всеми тратами.

## Что-то пошло не так

Напиши администратору. Это beta-версия, обратная связь помогает быстро чинить сценарии.

## Подсказка

`/help` — список всех команд
`/cancel` — выйти из любого диалога с ботом
